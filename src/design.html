My Design Projects

Saskatchewan Arlines Website Mockups

Date: November 2022
A fellow student and I created a weather app called 'F*ck The Weather'. It is a website application that allows the user to select a major Saskatchwan city and view weather information for the city selected. It uses crass language to describe the weather. Using the API open-meteo.com it displays current, weekly, and hourly weather information from one of the 5 Saskatchewan cities! This project was created for my advanced JavaScript class where we worked on await, async, JSON, and fetching data from APIS.

Tools Used

  • Adobe XD
  • HTML
  • CSS (Vanilla)
  • Javascript (Vanilla)
  • JSON
  • API: Open Metro
